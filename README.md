# 🍔 Food Delivery App

A full-stack **Food Delivery Web Application** that allows users to browse restaurants, view menus, add items to cart, and place orders online. Built with a modern web development stack for fast, responsive, and scalable performance.

---

## 🚀 Features

- 👤 User authentication (Login/Register)
- 🍽️ Browse restaurants and food items
- 🛒 Add to cart / remove items
- 📦 Place and track orders
- ❤️ Save favorite foods (optional feature)
- 📱 Fully responsive UI (mobile + desktop)
- 🧾 Order summary and checkout system

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS / CSS3
- React Router
- Axios

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

**Authentication:**
- JWT (JSON Web Token)
